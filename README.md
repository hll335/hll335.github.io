# HLL335 个人网站

一个用于展示自己、作品和联系方式的个人主页。访问根路径 [https://hll335.github.io](https://hll335.github.io) 就能直接看到首页，不需要手动输入 `/index.html`。

## 这是什么

这是一个已经做成单页结构的个人网站，适合当作个人名片、作品集首页或简历入口使用。页面重点放在清晰展示个人信息，让访问者可以快速了解你是谁、你会什么、你做过什么，以及如何联系你。

## 页面亮点

- 首页介绍：第一屏直接展示个人定位和一句话说明。
- 关于我：适合放自我介绍、背景经历和个人目标。
- 技能展示：用进度条快速呈现擅长方向。
- 作品展示：集中展示代表性项目和亮点。
- 联系入口：提供表单和外部链接，方便继续沟通。
- 主题切换：支持明暗模式，并记住上次选择。

## 页面结构

- [index.html](index.html)：网站入口，GitHub Pages 会默认加载它。
- [assets/css/style.css](assets/css/style.css)：页面样式，负责整体视觉和布局。
- [assets/js/main.js](assets/js/main.js)：交互脚本，负责主题切换、滚动进度和表单行为。

## 如何使用

1. 打开首页，先把姓名、简介、作品和联系方式换成你的真实信息。
2. 按需要增删作品卡片、技能项和时间线内容。
3. 如果想改样式，优先修改 [assets/css/style.css](assets/css/style.css)。
4. 如果想改交互，修改 [assets/js/main.js](assets/js/main.js)。

## 适合什么场景

- 个人主页
- 作品集首页
- 简历入口页
- 联系名片页
